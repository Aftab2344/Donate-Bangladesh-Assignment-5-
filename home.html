Important FAQ

What is the document object model(DOM)?

A:The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document (such as elements, attributes, and text). This allows programs and scripts to dynamically access, modify, add, or delete content and structure within a web page.

Q:How do You select an element from the DOM?

A: For selecting element there is four way to select . getelementById, getelementByClassNames, querySelectorAll,querySelector

Q: What is event delegation in the context of dom , and why it is useful?

A: Event delegation is a technique in the context of the DOM where a single event listener is attached to a parent element to manage events triggered by its child elements. Instead of adding event listeners to individual child elements, you delegate the event handling to a common ancestor. This takes advantage of event bubbling, where events propagate upwards from the target element to its parents, allowing the parent to "capture" the event

Q: How do you manipulate on elements attributes and styles using the DOM?

A: To manipulate attributes and styles in the DOM, use getAttribute(), setAttribute(), removeAttribute(), and hasAttribute() to access or modify element attributes like src, href, or alt. For styling, use the style property to modify inline CSS (e.g., element.style.backgroundColor = "blue"), or manage predefined styles using classList.add(), classList.remove(), and classList.toggle() to add, remove, or toggle CSS classes on elements efficiently. This allows dynamic control over both attributes and styles in response to user actions..
